# Slave Neet Blog

なんかいい感じにブログ書きたい気分になったので作り始めたものです。

## 仕様的なもの

NextjsのSSG(静的サイトジェネレーター)を利用してMicroCMSからコンテンツを取得していい感じにデプロイします。  
そのため、Cloudflare Pagesのようなサービスで運用でき、サーバー側でレンダリングするような処理は除外するように設計しています。
